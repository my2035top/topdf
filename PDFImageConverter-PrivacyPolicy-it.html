Informativa sulla Privacy

Ultimo aggiornamento: 15 maggio 2026

Riepilogo: PDF Image Converter NON raccoglie, memorizza né trasmette alcun dato personale. L'app funziona al 100% localmente sul tuo dispositivo e non si connette mai a nessuna rete.

1. Raccolta Dati

PDF Image Converter non raccoglie, acquisisce né registra alcuna informazione personale, dato utente o statistica di utilizzo. L'applicazione opera completamente offline.

2. Trasmissione Dati

L'applicazione non ha alcuna funzionalità di rete. Nessun dato viene mai inviato dal tuo dispositivo a nessun server:

  • Nessuna telemetria o analisi dell'utilizzo
  • Nessun rapporto sugli arresti anomali inviato a terze parti
  • Nessun account utente o autenticazione
  • Nessun archivio cloud o sincronizzazione

3. Elaborazione Locale dei File

Tutta la conversione da PDF a immagini viene eseguita localmente sul tuo Mac utilizzando il framework PDFKit integrato di Apple. I tuoi file PDF e le immagini convertite rimangono esclusivamente sul tuo dispositivo. Nessun file viene caricato, copiato o trasmesso a un server remoto.

4. Servizi di Terze Parti

PDF Image Converter non si integra con alcun servizio, SDK o framework di terze parti che raccoglie dati utente.

5. Autorizzazioni

L'app richiede l'accesso solo ai file che selezioni esplicitamente tramite la finestra di dialogo di selezione file di macOS. Non accede a file o dati oltre a quelli che scegli di convertire.

6. Privacy dei Minori

Non raccogliendo alcun dato, questa applicazione non pone alcun rischio per la privacy dei minori.
