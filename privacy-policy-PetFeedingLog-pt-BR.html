Política de Privacidade

Última atualização: 13 de maio de 2026

Obrigado por usar o Registro de Alimentação (doravante denominado "o Aplicativo"). Valorizamos profundamente sua privacidade, e esta Política de Privacidade explica de forma clara e transparente como o Aplicativo lida com suas informações.

🔒 Compromisso fundamental: Este aplicativo não coleta, envia ou compartilha nenhum dos seus dados pessoais.

1. Quais informações coletamos

Não coletamos nenhuma informação.

Este Aplicativo não possui sistema de registro de conta, não possui funcionalidade de login de usuário e não utiliza nenhum SDK de análise ou rastreamento de terceiros. Você não precisa fornecer nenhuma informação pessoal para usar este Aplicativo.

2. Uso de permissões

Este Aplicativo não solicita nenhuma permissão do sistema (como Fotos, Câmera, Localização, etc.). Todos os dados são armazenados exclusivamente no armazenamento local do seu dispositivo.

Nós nunca iremos:

  • Enviar seus dados para qualquer servidor
  • Compartilhar seus dados com terceiros
  • Usar seus dados para fins diferentes da funcionalidade principal

Todo o processamento de dados é realizado inteiramente no seu dispositivo.

3. Armazenamento de dados

As informações dos pets e os registros de alimentação gerados por este Aplicativo são armazenados exclusivamente no armazenamento local do seu dispositivo (arquivos JSON no diretório Documents).

Este Aplicativo não lê, verifica, envia ou analisa quaisquer outros arquivos ou dados no seu dispositivo.

Ao excluir o Aplicativo, todos os dados internos serão removidos simultaneamente.

4. Serviços de terceiros

Este Aplicativo não utiliza nenhum serviço de terceiros.

Especificamente, este Aplicativo não inclui:

  • SDKs de publicidade de terceiros
  • Ferramentas de análise de terceiros
  • Ferramentas de relatório de falhas de terceiros
  • SDKs de compartilhamento social de terceiros
  • Qualquer forma de rastreamento de comportamento do usuário

5. Segurança dos dados

Como este Aplicativo não coleta nenhum dado do usuário:

  • Não há armazenamento de dados do usuário no servidor
  • Não há riscos de segurança durante a transmissão de dados
  • Não há possibilidade de vazamento de dados

Seus dados são armazenados localmente no seu dispositivo e protegidos pelos mecanismos de segurança no nível do sistema do seu dispositivo (sandbox do iOS, criptografia de arquivos).

6. Privacidade infantil

Este Aplicativo não foi projetado para crianças menores de 13 anos. Como este Aplicativo não coleta nenhuma informação pessoal, não podemos identificar a idade dos usuários. Se pais ou responsáveis descobrirem que uma criança está usando este Aplicativo, não há necessidade de se preocupar com a coleta de dados, pois este Aplicativo não coleta nenhum dado.

7. Seus direitos

Como este Aplicativo não coleta nenhum dado pessoal, você naturalmente desfruta de proteção completa de privacidade:

  • Não é necessário solicitar acesso ou exportação de dados (pois nenhum dado é coletado)
  • Não é necessário solicitar exclusão de dados (pois nenhum dado é armazenado em servidores)
  • Desinstalar o Aplicativo removerá todos os dados internos

8. Alterações nesta política de privacidade

Se esta Política de Privacidade for alterada, iremos atualizá-la nesta página. Como este Aplicativo não coleta dados do usuário, não podemos notificá-lo por e-mail ou outros meios. Recomendamos verificar esta página periodicamente.
