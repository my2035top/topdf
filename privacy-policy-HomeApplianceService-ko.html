최종 업데이트: 2025년 5월 21일

「가전 AS」(이하 "본 앱")를 이용해 주셔서 감사합니다. 우리는 귀하의 개인정보 보호의 중요성을 깊이 이해하고 있으며, 본 개인정보 처리방침은 본 앱이 귀하의 정보를 어떻게 처리하는지 명확하고 투명하게 설명합니다.

🔒 핵심 약속: 본 앱은 귀하의 개인 데이터를 수집, 업로드 또는 공유하지 않습니다.

1. 수집하는 정보

당사는 어떠한 정보도 수집하지 않습니다.

본 앱에는 계정 등록 시스템, 사용자 로그인 기능이 없으며, 제3자 통계 또는 분석 SDK를 사용하지 않습니다. 본 앱을 사용할 때 개인정보를 제공할 필요가 없습니다. 귀하의 모든 데이터는 귀하의 기기에 로컬로 유지되며, 당사는 이 정보에 접근, 읽기, 수집하지 않습니다.

2. 권한 사용

본 앱은 핵심 기능 구현을 위해 오직 귀하의 기기의 일부 권한에 접근해야 할 수 있습니다:

  • 네트워크 권한: 가전제품 애프터서비스 정보 조회에 사용
  • 알림 권한: 애프터서비스 알림을 보내는 데 사용

당사는 다음을 하지 않습니다:

  • 귀하의 데이터를 서버에 업로드
  • 귀하의 데이터를 제3자와 공유
  • 핵심 기능 이외의 목적으로 귀하의 데이터 사용

모든 데이터 처리는 귀하의 기기에서 로컬로 수행됩니다.

3. 데이터 저장

본 앱에서 생성된 모든 데이터는 귀하의 기기 로컬 저장소에만 저장됩니다.

본 앱은 귀하의 기기에 있는 다른 파일을 읽거나, 스캔하거나, 업로드하거나, 분석하지 않습니다. 모든 데이터는 귀하의 완전한 통제 하에 있으며, 앱을 제거하여 언제든지 모든 데이터를 삭제할 수 있습니다.

4. 제3자 서비스

본 앱은 어떠한 제3자 서비스도 사용하지 않습니다.

구체적으로, 본 앱에는 다음이 포함되어 있지 않습니다:

  • 제3자 광고 SDK
  • 제3자 데이터 분석 도구
  • 제3자 충돌 보고 도구
  • 제3자 소셜 공유 SDK
  • 어떠한 형태의 사용자 행동 추적

5. 데이터 저장 및 보안

본 앱은 사용자 데이터를 수집하지 않으므로:

  • 사용자 데이터의 서버 저장이 존재하지 않습니다
  • 데이터 전송 중 보안 위험이 없습니다
  • 데이터 유출 가능성이 없습니다

귀하의 데이터는 귀하의 기기에 로컬로 저장되며, 그 보안은 귀하의 기기 시스템에 의해 보호됩니다.
