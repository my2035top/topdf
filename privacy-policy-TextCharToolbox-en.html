Privacy Policy

Last Updated: May 12, 2026

Thank you for using "TextCharToolbox" (hereinafter referred to as "the App"). We deeply value your privacy, and this Privacy Policy clearly and transparently explains how the App handles your information.

🔒 Core Commitment: The App does not collect, upload, or share any of your personal data.

1. What Information We Collect

We do not collect any information.

The App is a fully offline tool. It has no account registration system, no user login functionality, and does not use any third-party analytics or tracking SDKs. You are not required to provide any personal information to use the App. All text processing is done entirely on your device locally.

2. App Features

The App provides the following completely offline features:

  • Character Count (Chinese/English/Digits/Spaces/Punctuation/Total/Lines)
  • Text Formatting (Remove Spaces, Remove Line Breaks, Remove Both)
  • Case Conversion (UPPERCASE, lowercase, Capitalize)
  • Chinese Simplification/Traditional Conversion (Bidirectional)
  • Special Symbol Library (12 categories, 300+ symbols, tap to copy)

All features work without an internet connection and run entirely on your device.

3. Data Storage & Security

All data generated by the App is stored only in your device's local storage. Specifically:

  • Text content entered in any tool is stored only in the device's UserDefaults
  • App settings (language preference, theme mode) are stored only locally
  • No data is uploaded to any server
  • No data is shared with any third party

Uninstalling the App will clear all local data immediately.

4. Third-Party Services

The App does not use any third-party services.

Specifically, the App does not contain:

  • Third-party advertising SDKs
  • Third-party analytics tools (e.g., Google Analytics, Firebase)
  • Third-party crash reporting tools (e.g., Crashlytics)
  • Third-party social sharing SDKs
  • Any form of user behavior tracking
  • Any network request code

5. Children's Privacy

The App is not designed for children under 13 years of age. Since the App does not collect any personal information, we cannot determine the age of users. If parents or guardians discover that a child is using the App, there is no need to worry about data collection, as the App does not collect any data.

6. Your Rights

Since the App does not collect any personal data, you naturally enjoy complete privacy protection:

  • No need to request data access or export (as no data is collected)
  • No need to request data deletion (as no data is stored remotely)
  • Uninstalling the App will clear all local data

7. Changes to This Privacy Policy

If there are any changes to this Privacy Policy, we will update this page. Since the App does not collect user data, we cannot notify you via email or other means. We recommend checking this page periodically.
