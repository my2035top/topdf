Datenschutzrichtlinie

Letzte Aktualisierung: 15. Mai 2026

Zusammenfassung: PDF Image Converter sammelt, speichert oder überträgt KEINE personenbezogenen Daten. Die App läuft zu 100 % lokal auf Ihrem Gerät und stellt niemals eine Netzwerkverbindung her.

1. Datenerfassung

PDF Image Converter sammelt, erfasst oder protokolliert keinerlei persönliche Informationen, Benutzerdaten oder Nutzungsstatistiken. Die Anwendung arbeitet vollständig offline.

2. Datenübertragung

Die Anwendung verfügt über keine Netzwerkfunktionalität. Es werden niemals Daten von Ihrem Gerät an einen Server gesendet, einschließlich:

  • Keine Telemetrie oder Nutzungsanalytik
  • Keine Absturzberichte an Dritte
  • Keine Benutzerkonten oder Authentifizierung
  • Keine Cloud-Speicherung oder Synchronisierung

3. Lokale Dateiverarbeitung

Die gesamte PDF-zu-Bild-Konvertierung wird lokal auf Ihrem Mac unter Verwendung des integrierten PDFKit-Frameworks von Apple durchgeführt. Ihre PDF-Dateien und konvertierten Bilder verbleiben ausschließlich auf Ihrem Gerät. Keine Dateien werden auf einen entfernten Server hochgeladen, kopiert oder übertragen.

4. Dienste von Drittanbietern

PDF Image Converter ist nicht in Dienste von Drittanbietern, SDKs oder Frameworks integriert, die Benutzerdaten erfassen.

5. Berechtigungen

Die App fordert nur Zugriff auf Dateien an, die Sie ausdrücklich über den macOS-Dateiauswahldialog auswählen. Sie greift nicht auf Dateien oder Daten zu, die über das hinausgehen, was Sie zur Konvertierung auswählen.

6. Datenschutz für Kinder

Da wir keine Daten erfassen, stellt diese Anwendung kein Risiko für die Privatsphäre von Kindern dar.

7. Änderungen dieser Richtlinie

Wenn wir Änderungen an dieser Datenschutzrichtlinie vornehmen, aktualisieren wir das Datum der „Letzten Aktualisierung" am oberen Rand dieser Seite.
