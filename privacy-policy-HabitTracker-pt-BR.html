Política de Privacidade

Última atualização: 10 de junho de 2026

🔒 Compromisso Fundamental: Este aplicativo não coleta, carrega ou compartilha nenhum dos seus dados pessoais.

1. Quais informações coletamos

O HabitTracker não coleta, reúne ou armazena nenhuma informação pessoal. O aplicativo funciona completamente offline no seu dispositivo. Não coletamos nomes, endereços de e-mail, dados de localização, estatísticas de uso ou qualquer outra informação pessoalmente identificável. Todos os dados que você cria (como hábitos, lembretes e registros de progresso) existem exclusivamente no seu dispositivo. Não temos acesso a esses dados e eles nunca saem do seu dispositivo.

2. Uso de permissões

O HabitTracker pode solicitar as seguintes permissões exclusivamente para a funcionalidade principal. Todas as permissões são usadas apenas localmente:

  • Notificações: Usadas para exibir lembretes para seus hábitos. Todas as notificações são agendadas e processadas localmente no seu dispositivo. Nenhum dado é enviado para servidores externos.
  • Armazenamento local: Usado para salvar seus dados de hábitos, configurações e preferências no seu dispositivo. Todos os dados são armazenados exclusivamente localmente.

3. Armazenamento de dados

Todos os dados gerados ou armazenados pelo HabitTracker permanecem exclusivamente no seu dispositivo. O aplicativo usa o armazenamento local e o banco de dados local do seu dispositivo para manter seus dados de hábitos, configurações e preferências. Não há sincronização em nuvem, armazenamento no servidor ou transmissão externa de dados de qualquer tipo. Se você desinstalar o aplicativo, todos os dados associados serão permanentemente excluídos do seu dispositivo.

4. Serviços de terceiros

O HabitTracker não utiliza serviços de terceiros, análises, redes de publicidade ou tecnologias de rastreamento. O aplicativo é totalmente independente e funciona sem dependências externas que possam coletar ou processar dados do usuário.

5. Armazenamento e segurança de dados

Como todos os dados são armazenados exclusivamente no seu dispositivo, a segurança dos seus dados está em suas mãos. Recomendamos usar os recursos de segurança integrados do seu dispositivo, como bloqueio de tela, criptografia do dispositivo e manter seu sistema operacional atualizado para proteger seus dados. O aplicativo em si não implementa mecanismos de criptografia adicionais além dos fornecidos pelo sistema operacional.

6. Privacidade das crianças

O HabitTracker não coleta intencionalmente dados pessoais de crianças menores de 13 anos. Como o aplicativo não coleta nenhum dado pessoal dos usuários, ele é seguro para uso por pessoas de qualquer idade. Se você tiver dúvidas ou preocupações sobre a privacidade das crianças, entre em contato conosco.

7. Seus direitos

Como o HabitTracker não coleta ou armazena nenhum dado pessoal, não há dados pessoais para acessar, modificar ou excluir. Você tem controle total sobre todos os seus dados, pois eles permanecem exclusivamente no seu dispositivo. Você pode excluir seus dados a qualquer momento limpando os dados do aplicativo nas configurações do dispositivo ou desinstalando o aplicativo.

8. Alterações nesta política de privacidade

Podemos atualizar esta política de privacidade periodicamente. Notificaremos você sobre alterações publicando a nova política de privacidade nesta página e atualizando a data de "Última atualização" acima. Recomendamos que você revise esta política de privacidade periodicamente para verificar alterações. As alterações nesta política de privacidade entram em vigor quando publicadas nesta página.
