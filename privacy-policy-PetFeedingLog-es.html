Política de Privacidad

Última actualización: 13 de mayo de 2026

Gracias por usar Registro de Alimentación (en adelante, «la Aplicación»). Valoramos profundamente su privacidad, y esta Política de Privacidad explica de forma clara y transparente cómo la Aplicación maneja su información.

🔒 Compromiso fundamental: Esta aplicación no recopila, sube ni comparte ninguno de sus datos personales.

1. Qué información recopilamos

No recopilamos ninguna información.

Esta aplicación no tiene sistema de registro de cuentas, ni función de inicio de sesión, y no utiliza ningún SDK de análisis o seguimiento de terceros. No es necesario proporcionar ninguna información personal para usar esta aplicación.

2. Uso de permisos

Esta aplicación no solicita ningún permiso del sistema (como fotos, cámara, ubicación, etc.). Todos los datos se almacenan únicamente en el almacenamiento local de su dispositivo.

Nunca realizaremos las siguientes acciones:

  • Subir sus datos a ningún servidor
  • Compartir sus datos con terceros
  • Usar sus datos para fines distintos de la funcionalidad principal

Todo el procesamiento de datos se realiza completamente en su dispositivo.

3. Almacenamiento de datos

La información de mascotas y los registros de alimentación generados por esta aplicación se almacenan únicamente en el almacenamiento local de su dispositivo (archivos JSON en el directorio Documents).

Esta aplicación no lee, escanea, sube ni analiza ningún otro archivo o dato en su dispositivo.

Al eliminar la aplicación, se borrarán simultáneamente todos los datos dentro de la misma.

4. Servicios de terceros

Esta aplicación no utiliza ningún servicio de terceros.

En concreto, esta aplicación no incluye:

  • SDK de publicidad de terceros
  • Herramientas de análisis de terceros
  • Herramientas de informes de fallos de terceros
  • SDK de intercambio social de terceros
  • Ningún tipo de seguimiento del comportamiento del usuario

5. Seguridad de datos

Dado que esta aplicación no recopila ningún dato de usuario:

  • No existe almacenamiento de datos de usuario en servidores
  • No hay riesgos de seguridad durante la transmisión de datos
  • No existe posibilidad de fuga de datos

Sus datos se almacenan localmente en su dispositivo y están protegidos por los mecanismos de seguridad a nivel del sistema de su dispositivo (sandboxing de iOS, cifrado de archivos).

6. Privacidad infantil

Esta aplicación no está diseñada para menores de 13 años. Dado que esta aplicación no recopila ninguna información personal, no podemos identificar la edad de los usuarios. Si los padres o tutores descubren que un menor está usando esta aplicación, no hay motivo de preocupación sobre la recopilación de datos, ya que esta aplicación no recopila ningún dato.

7. Sus derechos

Dado que esta aplicación no recopila ningún dato personal, usted disfruta naturalmente de una protección completa de la privacidad:

  • No es necesario solicitar acceso o exportación de datos (ya que no se recopilan datos)
  • No es necesario solicitar la eliminación de datos (ya que no se almacenan datos en servidores)
  • Desinstalar la aplicación eliminará todos los datos dentro de la misma

8. Cambios en esta política

Si esta Política de Privacidad cambia, la actualizaremos en esta página. Dado que esta aplicación no recopila datos de usuario, no podemos notificarle por correo electrónico u otros medios. Le recomendamos consultar esta página periódicamente.
